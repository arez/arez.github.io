001package arez.annotations;
002
003/**
004 * Enum that controls how the component interacts with the injection framework if at all.
005 * Arez currently assumes that <a href="https://google.github.io/dagger/">Dagger2</a> is used
006 * as the injection framework. The artifacts that are generated when injections is enable
007 * vary based on the configuration of the component and whether the component has a method annotated
008 * with {@link PostConstruct}. This is an extremely rough part of the toolkit which requires excessive
009 * work by the user. It is likely that in the future all of this will be automated away.
010 *
011 * <ul>
012 * <li>
013 * A component configured with {@link #CONSUME} and either has no {@link PostConstruct} annotated method
014 * and no {@link Observe} annotated methods and no {@link Memoize} with keep alive set to <code>true</code>
015 * or only uses constructor injection will have the <code>javax.inject.Inject</code> annotation added to
016 * the constructor.
017 * </li>
018 * <li>
019 * A component configured with {@link #PROVIDE} and either has no {@link PostConstruct} annotated method
020 * and no {@link Observe} annotated methods and no {@link Memoize} with keep alive set to <code>true</code>
021 * or only uses constructor injection will have the <code>javax.inject.Inject</code> annotation added to
022 * the constructor and a Dagger2 module created named <code>[MyComponent]DaggerModule</code> which must be
023 * added to the desired component.
024 * </li>
025 * <li>
026 * A component configured with {@link #CONSUME} and a {@link PostConstruct} annotated method and has
027 * non-constructor based injection will have an interface named <code>[MyComponent]DaggerComponentExtension</code>
028 * created that the desired component must extend. The extension has a method named <code>bind[MyComponent]()</code>
029 * that must be invoked before any instances of the arez component can be created.
030 * </li>
031 * <li>
032 * A component configured with {@link #PROVIDE} and a {@link PostConstruct} annotated method and has
033 * non-constructor based injection will have an interface named <code>[MyComponent]DaggerComponentExtension</code>
034 * created that the desired component must extend. The extension has a method named <code>bind[MyComponent]()</code>
035 * that must be invoked before any instances of the arez component can be created. The extension also contains a dagger
036 * module named <code>[MyComponent]DaggerComponentExtension.DaggerModule</code> that MUST be added to the
037 * component manually.
038 * </li>
039 * </ul>
040 */
041public enum InjectMode
042{
043  /**
044   * InjectMode is disabled. The component can not be created by the injection framework.
045   * It is an error if the injections is disabled and the component has <code>javax.inject.Inject</code>
046   * annotations present.
047   */
048  NONE,
049  /**
050   * The component must be created and injected by the injection framework. The component MUST have a single
051   * constructor otherwise the injection framework will not know how to create the component.
052   */
053  CONSUME,
054  /**
055   * This enum has the same characteristics as {@link #CONSUME} and may also be injected
056   * into other components managed by the injection framework.
057   */
058  PROVIDE,
059  /**
060   * Allow the annotation processor to determine whether injection should be enabled.
061   * If any fields or methods in the component or any parent type has an <code>javax.inject.Inject</code>
062   * annotation OR the class has an annotation that is itself annotated with the <code>javax.inject.Scope</code>
063   * annotation then the component will be assumed to be {@link #PROVIDE} otherwise the value {@link #NONE}
064   * is assumed.
065   */
066  AUTODETECT
067}
